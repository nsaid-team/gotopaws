This page will have a list of type of pets (cats, dogs, birds), using the attributes from pets.json dataset.
The list can be sorted using any columns attribute
* Pet Type Attribute: Dogs will have a link to dogs_list page, the same for cats and birds.
* City Attribute: Each city will be a link to a City detail page what is coming from City model, the page already exist on cities model part
* Shelter Attribute: Each shelter has a link to a shelter_detail page coming from shelters models, the page already exist on shelters pages
* All others attributes doesn't have links to any page
